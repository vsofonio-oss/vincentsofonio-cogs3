How to define a table using HTML?

tr

(Table Row): This element defines a row within an HTML table. All table cells, whether header or data cells, must be contained within a tr element.

th

(Table Header): This element defines a header cell within a table row. It is used to provide headings for columns or rows, semantically indicating the content of the corresponding cells. By default, th content is typically bolded and centered.

td

(Table Data): This element defines a standard data cell within a table row. It holds the actual data content of the table.

thead

(Table Head): This element groups the header content of a table. It contains one or more tr elements that serve as the table's header rows, providing context for the data in the tbody

tfoot

(Table Foot): This element groups the footer content of a table. It typically contains summary information, totals, or other data relevant to the entire table, often appearing at the bottom.

caption

(Table Caption): This element provides a descriptive caption or title for the entire table. It should be placed immediately after the opening table tag and offers a concise summary of the table's content, which is particularly beneficial for accessibility.

Difference between colspan and rowspan?

colspan merges table cells horizontally across multiple columns
rowspan merges cells vertically across multiple rows.

Difference between cells spacing and padding?

Cell padding adds space between a cell's border and its content
Cell spacing adds space between adjacent cells
