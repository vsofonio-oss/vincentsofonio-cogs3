How to use an ordered list?

In order to use an ordered list in HTML you use the command ol and then use comamnds li to signify the start of the list and the /li command to end the list.

How to use an unordered list?

In order to use an ordered list in HTML you use the command ul and then use the commands li to signify the start of the list and /li to end the list.

How to use an descriptive list?

In order to use an ordered list in HTML you use the command d and then use commands dt to signify the larger title and the command dd to signify the smaller text that goes under.

Description and definition for the following network protocols: FTP, HTTP, TCP/IP

FTP

File Transfer Protocol:Used to transfer files between computers over a network.

HTTP

HyperText Transfer Protocol:Used for transferring web pages and data on the internet.

TCP/IP
